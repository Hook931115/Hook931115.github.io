Lua系列-语法2

1.表达式

1.1 算数运算符

  • + 加
  • - 减
  • * 乘
  • / 除
  • ^ 指数
  • % 取余
  • - 负号

下面是一些简单的例子：

a = 3
b = 5
print(a + b)  -- 8
print(a - b)  -- -2
print(a * b)  -- 15
print(a / b)  -- 0.6
print(a ^ b)  -- 243
print(a % b)  -- 3
print(-a) -- -3

1.2 关系运算符

  • <：小于
  • >：大于
  • <=：小于等于
  • >=：大于等于
  • ~=：不等于
  • ==：等于

table、userdata和function,Lua中是以他们的引用来作比较。也就意味着只有当他们指向同一个对象时才是相等的。

1.3 逻辑运算符

  • and：与
  • or：或
  • not：非

对于and来说，当第一个操作数为假时，返回第一个操作数，否则返回第二个操作数。

对于or来说，当第一个操作数为真时，返回第一个操作数，否则返回第二个操作数。

这样使用会带来一些语法上的便利。比如我们在代码中有时候会需要判断一个值是不是为空，为空的话给他赋值，常规的写法是这样的：

if not x then x = v end

我们可以写成这样：

x = x or v

还有一种用法时类似于C语言中的条件表达式，比如我们要找出两个数中的最大值：

max = ((a) > (b)) ? a : b

在Lua中我们可以这样写：

max = (a > b) and a or b

1.4 字符串连接操作符

  • .. 字符串连接操作符

字符串连接用来连接两个字符串，当后一个为其他类型时会转为字符串，比如：

print("Hello"..2)

1.5 操作符优先级

|^|
||
|not # -|
|* / %|
|+ -|
|..|
|< > <= >= ~= ==|
|and|
|or|

除了指数操作符和连接操作符是右结合外，其他的操作符都是左结合。

优先级不用死记硬背，因为我们还有()。

1.6 table构造式

构造式是用来创建和初始化table的表达式，是Lua中特有的一种表达式。最简单的构造式是这样的：

t = {};

构造式还可以初始化一个记录风格的table：

a = {x = 10,y = 20}

在构造一个Lua的table的时候，如果我们不给它一个key，那么默认的key就是从1，2，3，4开始的，比如我们要申请一个有关星期的table：

t = {"Sunday","Monday","Tuesday","Wednesday,"Thursday","Friday","Saturday"}

这样的一个table它的构造应该时这样的：

t = {}
t[1] = "Sunday"
t[2] = "Monday"
...

这段代码看起来是不是很眼熟？是的，它其实就是我们以前使用过的数组。如果你在一个table中既想使用数组也想使用map，我们也可以这样来构造一个table：

t = {
    name = ""

}
