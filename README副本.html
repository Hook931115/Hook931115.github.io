TKL

TKL is a responsive design theme for Hexo.
Demo| Kieran’s Blog

###Install
Execute the following command and modify theme in _config.yml to TKL.

git clone https://github.com/SuperKieran/TKL.git

###Update
Execute the following command to update TKL.

cd themes/TKL
git pull
